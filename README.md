Overview
This repository contains the "main.py" script, which performs exploratory data analysis (EDA) on a health dataset stored in CSV format. It utilizes Python libraries like pandas, seaborn, and matplotlib to analyze and visualize the data.

Features
Loads and inspects health data.
Identifies numeric and categorical features in the dataset.
Visualizes the distribution of ages by blood type using a violin plot.
Displays a pie chart representing admission types by medical condition.
Requirements
Make sure to have the following Python libraries installed:

bash
pip install numpy pandas seaborn matplotlib

Usage
Place the health.csv file in the specified directory.

Run the script:

bash
python main.py

Observe the outputs in the console and the visualizations displayed.
